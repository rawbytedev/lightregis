# Light

This is a small self-hosted registration + group making system
groups aren't made randomly but made by connected user 
Each group has a single admin aka creator the creator is the only one who can make
changes such as exclusion rules, or removing members
Registration form is as follow:

- PublicFields:
 showFields(A list of field to show + their order e.g: FirstName, LastName)

- PrivateFields:
fields that are private => password of user if applicable

The form allow custom creation and form are build dynamically
